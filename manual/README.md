# Blog
## Markdown
### 忽略副标题
> 默认情况下每个标题都会自动添加到目录中。

使用`<!-- {docsify-ignore} -->`忽略一个特定的标题
```markdown
    # Header <!-- {docsify-ignore} -->
```
在页面的第一个标题上使用`<!-- {docsify-ignore-all} -->`忽略页面上所有标题
```markdown
    # Header <!-- {docsify-ignore} -->
```
